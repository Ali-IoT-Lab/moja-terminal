* deamon 守护程序
  * deamon.sh pm2进程守护脚本 功能：每隔1分钟检测一次客户端代码进程，如果进程不存在就通过pm2 启动
* handleLog 日志打包
  * clearLog.sh 日志清除脚本 功能：每隔一周清除一次 维持7天以内到日志文件数量
  * tarlog.sh  日志打包脚本 功能：每天打包一次日志
* useAge 监控数据采集
  * useAge.sh 监控数据采集脚本 功能：采集内存 cpu 磁盘使用率
